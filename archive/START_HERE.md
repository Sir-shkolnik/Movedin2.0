## 🎯 Quick Start (3 Steps)

### **1. Navigate to Frontend**
```bash
cd MovedinV3.0/frontend
```

### **2. Install Dependencies** (if not done)
```bash
npm install
```

### **3. Start Development Server**
```bash
npm run dev
```

### **4. Open Browser**
```
http://localhost:5175
```
